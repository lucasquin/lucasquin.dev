O que é o Git?

Git é um sistema de controle de versões distribuido, usado principalmente no desenvolvimento de software, mas pode ser usado para registrar o histórico de edições de qualquer tipo de arquivo. Muitos softwares e livros digitais são mantidos com o uso do Git e possuem seu código fonte hospedado em um servidor com suporte ao Git (Github, GitLab, SourceForge etc.) em que uma comunidade pode acessar e contribuir com o repósitorio.

Vamos deixar claro que Git e Github são completamente diferentes, então vamos esquecer o Github e focar apenas no Git.
