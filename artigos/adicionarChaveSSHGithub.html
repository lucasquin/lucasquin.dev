O que é o GIT?

GIT é um sistema de controle de versões distribuido, usado principalmente no desenvolvimento de software, mas pode ser usado para registrar o histórico de edições de qualquer tipo de arquivo. Muitos softwares e livros digitais são mantidos com o uso do GIT e possuem seu código fonte hospedado em um servidor GIT (Github, GitLab, SourceForge etc.) em que uma comunidade pode acessar e contribuir com o repósitorio.

Vamos deixar claro que GIT e Github são completamente diferentes, então vamos esquecer o Github e focar apenas no GIT.
