Adicionar chave SSH ao Github

Para adicionar uma chave SSH no Github é bem simples, primeiro precisamos criar um par de chaves rsa na nossa máquina com o comando: ssh-keygen -t rsa -b 4096 -C "seuemail@exemplo.com"

O par de chaves criados, ficam dentro do diretório ~/.ssh teremos 2 chaves SSH, uma chave pública id_rsa.pub e uma chave privada id_rsa
