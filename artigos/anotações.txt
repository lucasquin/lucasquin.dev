
<h1>
    Adicionar chave SSH ao Github
</h1>
<p>
    Para adicionar uma chave SSH no Github é bem simples, primeiro precisamos criar um par de chaves rsa na nossa
    máquina com o comando:
    <code>ssh-keygen -t rsa -b 4096 -C "seuemail@exemplo.com"</code>
</p>
<p>O par de chaves criados, ficam dentro do diretório <code>~/.ssh</code> teremos 2 chaves SSH, uma chave pública
    <code>id_rsa.pub</code> e uma chave privada <code>id_rsa</code></p>
