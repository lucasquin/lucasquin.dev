Parte 1: Introdução aos bancos de dados

Banco de dados e usuários de banco de dados

Um banco de dados é um conjunto de dados relacionados de forma lógica e coerente que representam fatos conhecidos e que possuem um significado implicito.

Banco de dados são impactantes onde são usados, pois representam aspectos do mundo real e seus dados são derivados com algum grau de interação de eventos do mundo real, às vezes chamado de minimundo ou de universo de discurso. O minimundo é projetado, construido e preenchido para uma finalidade especifica.

Para exemplificar o minimundo de um supermecado, os usuários podem realizar transações comerciais ou podem acontecer eventos: Uma cliente retira um produto do supermecado, o estoque diminui (transação comercial). Uma funcionaria é promovida, seu cargo é atualizado (evento). Para que o banco de dados seja preciso e confiavel o tempo todo, ele precisa ser um reflexo e verdadeiro que esse minimundo(aspectos do mundo real) representa o mais rápido possivel.
